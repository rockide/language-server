# Rockide

Rockide is a language server for Minecraft Bedrock Addon development. (WIP)

## Features

Supported language features for JSON files:

- Completions
- Go to definitions
- Symbol rename
- Embedded mcfunction & molang

Supported language features for Molang:

- Completions
- Go to definitions
- Hover
- Signature help
- Syntax highlighting

Supported language features for mcfunction:

- Completions
- Go to definitions
- Hover
- Signature help
- Syntax highlighting
